# 哔哩哔哩开放平台概述

[哔哩哔哩开放平台](https://openhome.bilibili.com/doc)为开发者提供丰富的 API 接口，支持多种业务场景的开发需求。主要功能包括：

- **账号授权**：

  - B 站账号授权
  - 账号绑定

- **用户管理**：

  - 获取用户公开信息
  - 用户数据统计

- **内容管理**：

  - 视频稿件发布、删除和查询
  - 专栏稿件管理

- **直播功能**：

  - 开播管理
  - 直播间消息长连

- **数据开放**：

  - 用户授权后的数据访问
  - 稿件数据分析

- **运营支持**：
  - 运营规范
  - 处罚规则
  - 帮助中心
